Hexo 评论系统配置

发表于2022-06-05|更新于2024-06-05|技术博客
|字数总计:602|阅读时长:2分钟

基本介绍

butterfly支持Disqus、Disqusjs、Livere、Gitalk、Valine、Waline、Utterances、Facebook Comments、Twikoo、Giscus、Remark42、Artalk评论系统。

常用评论系统对比：

评论系统 优点 缺点
Disqus 评论功能强大，支持多种语言，自带反垃圾评论功能 需要注册账号，评论内容需要经过审核
Valine 无需注册账号，评论内容无需经过审核，支持多种主题 评论功能相对简单，不支持反垃圾评论功能
Gitalk 基于 GitHub Issues，无需注册账号，评论内容无需经过审核 需要配置 GitHub 账号，评论内容与 GitHub Issues 绑定
Remark42 无需注册账号，评论内容无需经过审核，支持多种主题 评论功能相对简单，不支持反垃圾评论功能

配置Valine系统

注册Leancloud

评论系统放在Leancloud上，首先需要注册一个账号，然后创建一个应用（名字随意）；
leancloud官网

获取appid和appkey

进入应用->设置->应用凭证

修改主题配置文件

打开主题配置文件 搜索 valine，填入appid 和 appkey(只需配置这两项即可)

测试

保存配置文件，重新上传博客

后台管理

评论存储在leancloud，打开创建的应用，数据存储->结构化数据->Comment

注意事项

在 Hexo 中，评论功能默认在所有页面都开启，如果部分页面不需要评论功能，例如分类、标记页面。
