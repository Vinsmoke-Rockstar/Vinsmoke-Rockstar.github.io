数据类型：

列表—list[ ]

(有序集合，可随时添加和删除元素)

classmates=[‘Michael’,‘Bob’]
⭐用[]表示

使用append()函数可以在其末尾添加元素；使用pop()可以删除list最后一个元素；
使用pop(i)可以删除指定位置的元素；index()用来查找列表中的元素，并返回索引值

元组—tuple( )

⚠初始化后不能修改
⚠显示只有一个元素的Tuple时，需要添加一个逗号
元组可以看作是不可变的列表，用于保存无需修改的内容
⭐用()表示

🌞 tuple和list之间的差异

  • tuple是静态的，大小固定的；list是动态分配存储空间，可以自动扩容
  • list要存储指向元素的指针以及当前长度大小，需要占用更多的存储空间。

字典—dict{ }

使用键-值(key-value)存储，具有极快的查找速度

d = {‘Michael’:95,‘Bob’:75,‘Tracy’:85}

⭐用{}表示
将数据放入dict，除了初始化指定外，还可以通过key放入

d['Adam']=67

一个key对应一个value，多次对一个key放入value，后面的值会将前面的值取代

判断key是否存在：

  1. 通过in判断key是否存在

‘Thomas’ in d
False

  1. 通过dict提供的get()方法，若key不存在，可返回None，或者自己指定的value

d.get(‘Thomas’)
d.get(‘Thomas’,-1)
>>-1

get()函数

get(key,value)——key表示确定要分配的键，value表示拟定给键分配的初值

  • 若字典中存在key，则返回字典中key对应的value值；
  • 若字典中不存在key时，返回拟定的value；

get(key)——若未指定value值，且字典中不存在key时，返回None

1. 利用键来获取初值
直接利用dict[key]取值时，当字典中不存在该键时，会返回KeyError类型错误，
此时可以通过get()函数利用键取值dict.get(key)

2. 利用字典统计列表中元素出现次数

删除一个key，使用pop(key)，对应的value也会从dict中删除

与list相比，dict查找和插入的速度极快，不会随着key的增加而变慢；但是需要占用大量的内存，内存浪费多

☀dict的key必须是不可变对象
dict根据key来计算value的存储位置，如果每次计算相同的key得出不同结果，dict内部就完全混乱了。这种通过key计算位置的算法称为哈希算法(Hash)

集合是一个无序不重复元素的集，基本功能包括关系测试和消除重复元素。
集合对象支持union（联合）、intersection（交）、difference（差）和symmetric difference（对称差集）等数学运算。

特殊变量与私有变量

类似__xxx__这样的变量是特殊变量，比如__author__，__name__就是特殊变量(两边有下划线的为特殊变量)，可以被直接引用

类似_xxx和__xxx这样的函数或变量就是非公开的（private），不应该被直接引用，比如_abc，__abc等(单边有下划线为非公开变量)

  • extend和append

下划线的含义

单前导下划线：_var

单个下划线是一个Python命名约定，表示这个名称是供内部使用的，通常不由Python解释器强制执行，仅仅作为一种对程序员的提示。

单末尾下划线：var_

当一个变量最适合的名称已经被一个关键字所占用，像class或def这样的名称，不能用作python中的变量名称，此时可以通过附加一个下划线来解决命名冲突。

双前导下划线：__var

双下划线前缀会导致Python解释器重写属性名称，以避免子类中命令冲突，也叫名称修饰（name mangling）

解释器更改变量的名称，以便在类被扩展的时候不容易产生冲突。

双前导和末尾下划线：__var__

常用于__init__，__call__、__iter__、__next__这些方法中，

避免在自己的程序中使用以双下划线开头和结尾的名称，以避免与python语言产生冲突。

单下划线：_

将单个独立的下划线用作名字，用来表示某个变量是临时的或无关紧要的
