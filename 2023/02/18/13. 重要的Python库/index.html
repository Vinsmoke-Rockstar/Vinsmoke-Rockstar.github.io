13. 重要的Python库

发表于2023-02-18|更新于2024-05-31|学习记录
|字数总计:1.3k|阅读时长:4分钟

numpy(Numerical Python)

功能：

  • 快速高效的多维数组对象
  • 用于读写硬盘上基于数组的数据集的工具
  • 线性代数运算、傅里叶变换，以及随机数生成
  • 作为算法与库之间传递数据的容器

ndarray是一个通用的同构数据多维容器，所有元素必须是相同类型
每个数都有一个shape（表示各维度大小的元组）和一个dtype（用于说明数据类型）

创建ndarray

使用array函数，接受一切序列型的对象（包括数组），然后产生一个新的含有传入数据的Numpy数组

使用zeros和ones分别可以创建指定长度或形状的全0或全1数组
empty可创建一个没有任何具体数值的数组

排序

Numpy数组可通过sort方法就地排序

多维数组可在任何一个轴向上进行排序，只需将轴编号传给sort即可

唯一化及其它集合逻辑

np.unique用于找出数组中唯一值并返回已排序的结果

pandas

  • pandas提供了快速便捷处理结构化数据的大量数据结构和函数，含有使数据清洗和分析工具变得更快更简单的数据结构和操作工具

  • pandas专门为处理表格和混杂数据设计，Numpy更适合处理统一的数值数组数据

两个主要数据结构Series和DataFrame

Series

  • Series是一种类似于一维数组的对象，由一组数据以及一组与之相关的数据标签（即索引）组成。

  • Series字符串表现形式为：索引在左边，值在右边。

若数据存放在一个Python字典中，则可以直接通过这个字典来创建Series

  • Series会根据运算的索引标签自动对齐数据

Series的索引可通过幅值的方式就地修改

DataFrame

DataFrame是一个表格型的数据结构，含有一组有序的列，每列可以是不同的值类型（数值、字符串、布尔值等）。

DataFrame既有行索引，也有列索引。

创建DataFrame数据最常用的方法是直接传入一个由等长列表或NumPy数组组成的字典
DataFrame会自动加上索引，且全部列会被有序排列

head方法可以选取前五行

os模块

os就是“operating system”的缩写，顾名思义，os模块提供的就是各种 Python 程序与操作系统进行交互的接口。通过使用os模块，一方面可以方便地与操作系统进行交互，另一方面页可以极大增强代码的可移植性。如果该模块中相关功能出错，会抛出OSError异常或其子类异常。

os库提供通用的、基本的操作系统交互功能，包含几百个函数，分为

  • 路径操作：os.path字库，处理文件路径及信息
  • 进程管理：启动系统中其他程序
  • 环境参数：获得系统软硬件信息等环境参数

🦜
读写文件，建议使用内置函数open()；路径相关操作，建议使用os的子模块os.path；
逐行读取多个文件，建议使用fileinput模块；创建临时文件或路径，建议使用tempfile模块。
导入os模块时，不能将os模块解包导入，即不要使用from os import *来导入os模块，否则os.open()将会覆盖内置函数open()

os.walk()

os.walk()方法通过在目录树中游走输出在目录中的文件名，向上或者向下

语法格式如下：

  • top：需要遍历目录的地址，返回一个三元组（root，dirs，files）
    • root指的是当前遍历文件夹的绝对路径
    • dirs是一个list，内容是该文件夹中所有子文件的名称（仅一层，不包括子目录）
    • files是一个list，指该文件夹中所有文件的名称（不包括子目录）
  • topdown：可选，如果为True，则优先遍历top目录，否则优先遍历top的子目录
  • onerror：可选，需要一个callable对象，当walk需要异常时，会调用
  • followlinks：可选，如果为True，则会遍历目录下的快捷方式实际所指的目录，如果为False，则优先遍历top的子目录。
